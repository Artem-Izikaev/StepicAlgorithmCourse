Задача на программирование: лестница

Даны число $1 \le n \le 10^2$ ступенек лестницы и целые числа $-10^4 \le a_1, \ldots, a_n \le 10^4$, которыми помечены ступеньки. Найдите максимальную сумму, которую можно получить, идя по лестнице снизу вверх (от нулевой до $n$-й ступеньки), каждый раз поднимаясь на одну или две ступеньки.


Sample Input:
2
1 2

Sample Output:
3

Sample Input:
2
2 -1

Sample Output:
1

Sample Input:
3
-1 2 1

Sample Output:
3

Memory limit: 256 Mb
Time limit: 5s
