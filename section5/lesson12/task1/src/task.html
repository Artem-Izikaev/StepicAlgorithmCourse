Задача на программирование: расстояние редактирования


Вычислите расстояние редактирования двух данных непустых строк длины не более $10^2$, содержащих строчные буквы латинского алфавита.


Sample Input:
ab
ab

Sample Output:
0

Sample Input:
short
ports

Sample Output:
3

Memory limit: 256 Mb
Time limit: 2s
